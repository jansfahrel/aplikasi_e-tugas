# 🚀 Aplikasi E-Tugas

Aplikasi **E-Tugas** adalah sistem berbasis web yang dirancang untuk membantu pengelolaan dan pengumpulan tugas secara digital.
Dibangun menggunakan framework **Laravel**, aplikasi ini mempermudah interaksi antara admin dan pengguna dalam mengelola tugas secara efisien.

---

## ✨ Fitur Utama

* 🔐 Sistem Login & Register
* 🧑‍💼 Dashboard Admin
* 📤 Upload & Pengumpulan Tugas
* 📋 Manajemen Data Tugas
* 💬 Sistem Interaksi User & Admin
* 📊 Tampilan Modern & Responsif

---

## 🛠️ Teknologi yang Digunakan

* ⚙️ Laravel (PHP Framework)
* 🗄️ MySQL Database
* 🎨 HTML, CSS, JavaScript
* 🔌 Vite (Frontend Build Tool)

---

## 📥 Cara Instalasi

1. Clone repository:

   ```bash
   git clone https://github.com/jansfahrel/aplikasi_e-tugas.git
   ```

2. Masuk ke folder project:

   ```bash
   cd aplikasi_e-tugas
   ```

3. Install dependency:

   ```bash
   composer install
   ```

4. Copy file environment:

   ```bash
   cp .env.example .env
   ```

5. Generate key:

   ```bash
   php artisan key:generate
   ```

6. Setup database di file `.env`

7. Jalankan migrasi:

   ```bash
   php artisan migrate
   ```

8. Jalankan server:

   ```bash
   php artisan serve
   ```

---

## 📌 Catatan

* Pastikan XAMPP / Laragon sudah aktif
* Jangan upload file `.env` ke GitHub
* Gunakan PHP versi terbaru untuk performa maksimal
